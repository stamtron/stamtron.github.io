# Anastasios Stamoulakatos - Personal Website

Personal academic and professional website of **Anastasios (Tasos) Stamoulakatos**, Data Scientist at Satalia and PhD in Applied AI.

This site is built with [Jekyll](http://jekyllrb.com/) and serves as a portfolio for my publications, curriculum vitae, and blog posts.

## Content
- **About**: Professional bio and current role.
- **CV**: Link to LinkedIn profile and archived PDF CV.
- **Publications**: Details of PhD and Master's theses.
- **Posts**: Links to Medium blog.

## Running Locally

1.  Make sure you have Ruby and Bundler installed.
2.  Install dependencies:
    ```bash
    bundle install
    ```
3.  Serve the site:
    ```bash
    bundle exec jekyll serve
    ```
